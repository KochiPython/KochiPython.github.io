KochiPython March-2015 Meetup Report

// under python meetup report

The fourth meetup was held at Zyxware Technologies on 7th March, 2015. Total attendence was above 30 including the speakers.

We started at 3:30PM with the session on Logging in python for large applications by Fayaz Yusuf Khan. Fayaz started with the basic logging levels and libraries like logbook available in python. He also talked about error reporting methods in various frameworks and his experiences with the advanced logging tools like sentry.

Slide

Logging in Python for large applications from Fayaz Yusuf Khan

We resumed the sessions after the refreshment break at 4:30 PM.

Our second session was by Lentin Joseph on Introduction to computer vision using openCV and python. He started with different types of computer vision and explained those with real time examples. He also explained basic usage of openCV in computer vision with the help of some sample codes and practical examples.

Before winding up, Vimal Joseph from Zyxware Technologies delivered a talk on some interesting facts about python.

The Meetup ended at 6PM.
