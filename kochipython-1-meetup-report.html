KochiPython #1 Meetup Report

// under python meetup report

First meetup was an exiting experience. There were 43 participants including presenters.

Started at 4.35, Mathews introduced Kochi Python meetup. Had an icebreaking session. Everyone introduced themeselves.

Rahul Ramesh was the first speaker. Introduced Vagrant, for easy setup of Development workflows, Docker for the production deployment. Demonstrated simply how they help to reduce devleopment/production time in python webapp development.

Myself took a flash session on how Ansible could be used for automating production deployment and maintance.

Slide

Nuventure had arranged snacks cookies for the tea break. And we resumed the sessions after 15 minutes.

Faris Aboobacker, COO of Nuventure, gave a small intro about Nuventure Technology solutions, how nuventure contributing to the python community.

In a quick sesion, Sreekanth exihited how anyone could write a blog post on KochiPython blog using Pelican framework.

Slide

Arun Jayan demoed various libraries in python for hardware interaction. He explained very well on PySerial, Microhope, PyVoltmeter and his experiences as a developer at ExpEYES hardwares.
